# FashionSense ML

## Description
Docker container hosting our machine learning stuff.

## Install
* [install Docker](https://docs.docker.com/engine/installation/)

## Usage
* to run with Docker
    * first build everything with Docker
    ```
    $ docker build -t ml .
    ```
    * to run with Docker
    ```
    $ docker run -p 5000:5000 ml
    ```
